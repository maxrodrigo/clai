# Self-Refine — improve output quality through self-critique and revision

Produce an initial response, then critique it for gaps, errors, or weaknesses, then produce an improved final response that addresses those issues.

This consistently improves output quality for writing, code generation, and any task where a first draft benefits from review — catching issues that the initial pass overlooked.

Format your response as:
**Draft:** [initial answer]

**Critique:** [what is weak, incomplete, or wrong about the draft]

**Refined answer:** [improved response addressing the critique]
